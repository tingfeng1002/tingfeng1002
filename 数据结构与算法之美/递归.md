## 递归函数示范
Recursion

```go
func f(n int) int {
	if n == 1 {
		return 1
	}
	return f(n-1) + 1
}

```

## 满足递归的三个条件
1、求解问题可以拆解成几个子问题的解决
2、子问题的求解思路是一致的
3、存在终止条件


## 举例
n 级的台阶 ，每一步可以可以是走一台阶 和走俩台阶 总共多少种走法，<br/>
1. 分析：假设 第一级有1种类走法 那么总共走法是 f(n-1)+f(1) 类推 n 级走法是f(n) = f(n-1) + f(n-2)
2. 找到终止条件 f(1) =1 第一节别就不需要走 只有1 种走法，实际是 1 层阶级确实是2 种，所以需要f(0) =1 ，但是 实际上不存在0 所以更改下终止条件 f(2) =2

```go 

func f(n int) int {
    if n== 1{
        return 1
    }  
    if n==2{
        return 2
    }
    
    return f(n-1) + f(n-2)
}
```
