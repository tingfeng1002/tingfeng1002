# 布隆过滤器

将数据经过多个哈希函数 存入位图的多个位置 来确保唯一性

## 数据的误判

假设 [] [] [] [] [] [] [] [] []分别对应的位数，对于数字 123 分别进行三次hash之后得到 2，4，7
<br/>
将数据的位置更改为1, 结果是
```
[0] [1] [0] [1] [0] [0] [1] [0] [0]
```
对于数字 456 分别进行三次hash 之后位置时 1，3，5 结果是
```
[1] [1] [3] [1] [5] [0] [1] [0] [0]
```
然后判断数字 567 是否存在 三次hash 之后的位置分别是 1，5 ，7 判断结果是存在，但实际数据并没存入
